# IntentProject

Es un ejemplo de Intents explícitos, se compone de 3 actividades. Actividad principal y dos actividades más que serán llamadas desde la primera a través de intents explícitos. La actividad principal pasa un texto como parámetro a las otras 2 actividades. Las otras dos actividades devuelven un texto a la actividad principal y un código de resultado.
